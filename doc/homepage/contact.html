If you'd like to contact us to report a bug or submit a patch, please use the tracker on RubyForge.

If you'd like to ask questions about using Ruby/DBI, there is a ruby-dbi-users email list where those questions can be answered. Likewise, ruby-dbi-bugs is for discussing bug reports in detail, and ruby-dbi-devel is for discussing future development and management of the project.

If these avenues are not working for you for some reason, please feel free to contact Erik or Christopher directly. They will be glad to help!
